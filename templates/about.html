À Propos de l'Application

Cette application analyse les votes des députés pour identifier les plus similaires en fonction de leurs positions sur diverses lois. Les données proviennent des fichiers XML fournis par l'Assemblée Nationale.

Méthodologie

L'analyse utilise le clustering hiérarchique et la similarité cosinus pour regrouper les députés et identifier les plus proches par leurs votes.

Sources des Données

Les données de vote sont extraites des fichiers XML disponibles sur le site officiel de l'Assemblée Nationale.
